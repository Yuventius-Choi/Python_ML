# Python ML

이 Repository는 python+ML에 대한 내용을 정리하고자 만든 저장소입니다.<br>
모든 파일은 `.ipynb`으로 구성되어 있으며 해당 파일을 열기위해서는<br>
Jupyter Notebook이나 Colab을 사용하여 여시면 됩니다.

## 실행환경 (Window)
- OS: Windows 11 Home (x64)
- CPU: Intel(R) Core(TM) i7-8700 CPU @ 3.20GHz(3.19 GHz)
- RAM: 16GB
- GPU: NVIDIA GeForce RTX 2080 (8 GB)
- IDE: IntelliJ Idea Ultimate
    - SDK: IntelliJ Virtual Python 3.9
    - Plugin
        - Python
        - Jupyter Notebook

## 실행환경 (MacOS)
- OS: MacOS Tahoe 26.5.1
- CPU: M3 Max
- RAM: 36GB
- IDE: IntelliJ Idea Ultimate
    - SDK: conda local server
    - Plugin
        - Python

Python 패키지인 .venv는 포함되지 않습니다. (직접 받으셔야 합니다.)

## 경로 구성
- `./data`
    - 실습에 사용하는 데이터 파일을 저장하는 폴더입니다.
    - 예제 데이터셋이나 Kaggle 등에서 내려받은 데이터를 이곳에 보관합니다.
    - 데이터 파일은 용량이 크거나 저작권 문제가 있을 수 있으므로, Git에서는 제외되었습니다.
